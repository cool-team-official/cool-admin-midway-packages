### COOL-ADMIN

cool-admin一个很酷的后台权限管理系统，开源免费，模块化、插件化、极速开发CRUD，方便快速构建迭代后台管理系统

大数据、微服务、AI编码快速开发！！！


### 技术栈

- 后端：node.js midway.js koa.js mysql typescript
- 前端：vue.js element-plus jsx pinia vue-router
